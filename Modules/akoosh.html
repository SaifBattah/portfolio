AKOOSH — Cafe & Barbershop

A dual-brand web platform for two businesses sharing one space — a specialty cafe and a modern barbershop. Features a cinematic split-screen landing, full menu, appointment booking, bilingual Arabic/English content, location info, and a complete social hub.

Key Features
🎬
Split-Screen Landing
Cinematic full-viewport hero that expands toward whichever side the user hovers — cafe or barbershop — with smooth CSS transitions and brand-specific color themes.
☕
Cafe Menu
Full categorized menu with bilingual item cards (Arabic + English), pricing, descriptions, and a smooth ordering flow.
✂️
Barbershop Booking
Services grid with pricing, appointment booking form, barber profiles, and working hours per day.
🌐
Bilingual & Social
Full Arabic + English support with RTL handling. Social hub connecting TikTok, Instagram, Snapchat, WhatsApp, and Facebook.
Tech Stack
HTML CSS JavaScript Node.js Responsive Design Bilingual / RTL
